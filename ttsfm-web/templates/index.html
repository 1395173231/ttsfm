Quick Start

Getting Started

Install TTSFM and start generating speech from text with a few lines of code.

1. Installation

pip install ttsfm
Requires Python 3.8+

2. Basic Usage

from ttsfm import TTSClient

# No API keys needed - uses openai.fm free service
client = TTSClient()

# Generate speech for free
response = client.generate_speech(
    text="Hello, world! This is free TTS.",
    voice="alloy"
)

# Save to file
response.save_to_file("hello.wav")
print("Speech generated successfully!")
Synchronous client for simple use cases

3. Async Usage

from ttsfm import AsyncTTSClient

async with AsyncTTSClient() as client:
    response = await client.generate_speech(
        text="Hello, async world!",
        voice="nova"
    )
    response.save_to_file("hello_async.mp3")
Async client for high-performance applications

4. Batch Processing

from ttsfm import AsyncTTSClient, TTSRequest

requests = [
    TTSRequest(input="First text", voice="alloy"),
    TTSRequest(input="Second text", voice="echo")
]

async with AsyncTTSClient() as client:
    responses = await client.generate_speech_batch(requests)
